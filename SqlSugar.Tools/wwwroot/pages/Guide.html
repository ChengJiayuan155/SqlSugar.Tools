SqlSugar.Tools 使用教程（分层生成说明）
这里是工具内置说明，包含连接数据库、命名规范、注释/枚举/导航规则，以及分层(MVC)生成设计理念。 （更详细版本请查看仓库根目录 README.md）
  1. 首页点击 实体类生成
  2. 选择数据库类型并添加连接，点击 测试连接，然后 确定
  3. 左侧加载表，点击表即可预览右侧代码
  4. 支持保存单表/保存全部表到目录

类上

会生成 [SugarTable(\"表名\", TableDescription=\"表中文注释\")]

字段上

会生成 [SugarColumn(ColumnDescription=\"列中文注释\", IsPrimaryKey/IsIdentity/IsNullable...)]

非表字段/导航字段请使用 [SugarColumn(IsIgnore=true)]（增强版会自动生成）。

推荐外键字段命名：主表名 + Id，例如 UserId / RoleId。

识别优先级：真实 FK（数据库约束）＞ 逻辑外键（UserId 推断）＞ 兼容旧规则（可选）。

在列备注里写：

(Enum:0=禁用,1=启用)

即可稳定解析并生成对应枚举（增强中）。

建议分层：Model / DAL / BLL / Common / WebApi（可选 Auth）。

推荐使用 SqlSugarScope 并做依赖倒置（上层依赖接口，下层实现封装 CRUD，同步+异步）。
